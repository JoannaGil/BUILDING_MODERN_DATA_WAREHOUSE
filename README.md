# Data Warehouse and Analytics Project

**Welcome to the Data Warehouse and Analytics Project repository!** 🚀  

<p align="justify">
This repository presents a complete Data Warehouse and Analytics portfolio project, covering the full data lifecycle from raw data ingestion to analytical insights.
It demonstrates the implementation of a Bronze–Silver–Gold architecture, including data cleaning, standardization, integration, and dimensional modeling.
Every step of the process is fully documented, providing a clear and transparent view of the decisions, transformations, and methodologies used to build an analytics-ready data model.
</p>

---
## 🏗️ Data Architecture
<p align="justify">
This project follows a modern layered data architecture using the Bronze, Silver, and Gold approach. 
Each layer represents a stage in the data transformation process, ensuring data quality, consistency, 
and readiness for analytics.
</p>

<img width="1362" height="697" alt="Data Architecture" src="https://github.com/user-attachments/assets/f5ff9ecb-059b-40f6-8556-878e614c68ab" />


1. **Bronze Layer**: Stores raw data as-is from the source systems. Data is ingested from CSV Files into SQL Server Database.
2. **Silver Layer**: This layer includes data cleansing, standardization, and normalization processes to prepare data for analysis.
3. **Gold Layer**: Houses business-ready data modeled into a star schema required for reporting and analytics.
---
## 🎯 Project Overview

<p align="justify">
This project is designed to provide hands-on experience in building a complete Data Warehouse and Analytics solution. 
By working through each stage, the reader can understand not only how to implement the solution, but also the reasoning 
behind each decision in the data engineering process.
</p>

This project involves:

- **Data Architecture:** Learn how to design a modern data warehouse using the Medallion Architecture (Bronze, Silver, Gold), understanding the purpose and responsibilities of each layer.

- **ETL Pipelines:** Gain practical experience extracting, transforming, and loading data from raw sources, including handling data quality issues and ensuring consistency.

- **Data Modeling:** Understand how to design and implement fact and dimension tables, apply star schema principles, and use surrogate keys for scalable analytics.

- **Analytics & Reporting:** Explore how structured data enables efficient querying, and how to generate meaningful insights using SQL-based analysis.
